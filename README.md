## Run locally

1. Git clone project

2. Run back-end :
```cd devback```
```npm install```
```npm run start```

3. Run front-end :
```cd devfront```
```npm install```
```ng serve --open``` or ```ng serve --live-reload```

4. Access webpage : http://localhost:4200/home
